The Atbash cipher is another type of substitution cipher.
It entails replacing each letter in the plaintext with the corresponding letter if the alphabet were reversed.
   plaintext: ABCDEFGHIJKLMNOPQRSTUVWXYZ
   cipher : ZYXWVUTSRQPONMLKJIHGFEDCBA

For instance,
   plaintext: BUY ZOO
   encrypted: YFB ALL


Task: decrypt ENCRYPTED_TEXT.
